# Build your first program

In this tutorial you'll build a 4-day upper/lower program with A/B
variations — the same shape Cadence's author trains with — set your
starting weights, and run its first training day. It takes about fifteen
minutes, and by the end you'll have a program that progresses itself for
as long as you keep banking sessions.

You can do this on the web app or the iOS app; the steps are the same.
(If you'd rather start from a ready-made style and skip the building,
see [Start from a pre-programmed style](../how-to/start-from-a-style.md)
— you can come back here to learn what everything means.)

## 1. Create the program

Open **Settings** and find the **Programs** section. Tap **+ Add
program**, then choose **Blank program**.

Tap the new program to open its editor and give it a name — "Upper/Lower
A/B". Two settings matter here:

- **Focus** — choose **Strength**. This controls how aggressively weight
  is added and how close to your estimated max the program will push
  (see [Progression rules](../reference/progression-rules.md)). Pick
  Hypertrophy for more conservative increments at higher volume, or
  Maintain to hold weights steady.
- **Rounding** — leave at 5 lb unless your gym has smaller plates. Every
  prescribed weight snaps to this granularity.

Make sure the program is marked **Active** — the active program is what
the Today screen offers you.

## 2. Add the four days

In the program editor, tap **+ Add day** four times, then open each day
and name them: **Upper A**, **Lower A**, **Upper B**, **Lower B**. The
order you see is the order they'll rotate.

## 3. Add lifts to each day

Open **Upper A** and tap **+ Add lift**. Pick your press — say Overhead
Press. Each lift has three things to set:

- **Role** — one lift per day should be **Main** (the day is built around
  it, it rests longest); the rest are **Comp.** (complementary).
- **Rotation-1 base** — the weight week 1 is prescribed at. Week 1 asks
  for **5×5** at this weight, so pick something you can confidently do
  for five sets of five today. Honest and light beats ambitious: the
  program adds weight every clean cycle, so it will catch up to you fast.
- **Est. 1RM** — your rough one-rep max. Don't overthink it; the engine
  re-estimates it from every peak week you bank. It exists so the
  program knows how much headroom you have.

Add a second lift as **Comp.** (e.g. Incline DB Press), then repeat for
the other days — squat-based lifts on Lower A, hinge-based on Lower B,
so the A/B pairs don't repeat the same pattern back-to-back.

## 4. Add accessories

Still in the day editor, tap **+ Add accessory** and pick, say,
Chin-ups. Accessories progress by **rep range**, not by the wave:

- **Sets** and **Rep range** — e.g. 3 sets of 6–10.
- **Load step** — the weight added once you earn the top of the rep
  range. Leave it at **0 for bodyweight movements**: they climb in reps
  instead of load.

Two or three accessories per day is plenty to start.

## 5. Check your position

Back in the program editor, the **Rotation** control shows where you are
in the 4-week wave — week 1 (volume), 2 (load), 3 (peak), 4 (rest). A
new program starts at week 1, day 1, which is what you want. (This
control is also how you'd tell Cadence "I'm mid-cycle" if you migrated
from a notebook.)

## 6. Run the first day

Go to the **Today** screen. The card shows **Upper A** with every lift
and its prescribed weight/sets/reps already filled in. Tap **Start**.

Everything is pre-filled from the program — barbell lifts even get a
warmup ramp. Log your sets as you do them (tap to confirm, adjust reps
or weight only if reality differed), use the rest timer between sets,
and when you're done, tap **Bank it.**

That's the whole loop. Banking records your history, checks for PRs,
advances the program's day pointer, and progresses your accessories.

## What happens next

Keep banking days. When the wave reaches **week 3 (peak)**, the program
grades each main and complementary lift on what you actually did. After
**week 4 (rest)**, the cycle rolls over: clean lifts get a tapered
weight increase, misses hold or eventually trigger a −10% deload, and
week 1 of cycle 2 begins with new prescriptions. You never have to
decide what to lift — that's the point.
